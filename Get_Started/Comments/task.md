# Comments

C++ 里支持两种 comments (注释) 风格:
- Line Comments (行注释)
  - 常用于 Function 内部语句注释
  - i.e.
    ```c++
    // This is a line comment
    int a = 10;
    ```
- Block Comments (块注释)
  - 常用于 class 和 function 的文档说明, 置于 declaration (声明) 之前
  - i.e.
    ```c++
    /**
     * This is a block comment
     */
    class A {...};
    ```
