# Comments

C++ 里支持两种 Comments (注释) 风格:
- Line Comments (行注释)
  - 惯例用于 Function 内部语句注释
  - i.e.
    ```c++
    // This is a line comment
    ```
- Block Comments (块注释)
  - 惯例用于 Class 和 Function 的文档说明, 常置于声明前
  - i.e.
    ```c++
    /**
     * This is a block comment
     */
    ```

请将左侧代码中的**三处**说明文字用合适的风格进行注释～
