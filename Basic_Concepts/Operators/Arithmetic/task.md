# Arithmetic Operators

Arithmetic Operators (算数操作符) 支持基本的四则计算, 适用于 Integer Type 和 Floating Point Type.
它们的使用风格与传统的数学公式基本保持一致.

```c++
+a     // 取正, == a
-a     // 取负
a + b  // 加法
a - b  // 减法
a * b  // 乘法
a / b  // 除法
a % b  // 求余数
```

其中需要注意的是:
- `/`: 当两个 Operands 都是 Integer Type 时, 计算结果将向下取整
- `%`: 两个 Operands 必须是 Integer Type
